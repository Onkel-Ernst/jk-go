git-Befehle - Beispiele (im Terminal)
-------------------------------------
create a new repository on the command line:
echo "# jk-go" >> README.md
git init
git add README.md
git commit -m "first commit"
git branch -M main
git remote add origin https://github.com/Onkel-Ernst/jk-go.git
git push -u origin main

…or 
push an existing repository from the command line
git remote add origin https://github.com/Onkel-Ernst/jk-go.git
git branch -M main
git push -u origin main
